```python
# Assigning a string value to the variable
input = 'New Delhi is the capital of India'
```


```python
# Split the string into words by calling split() built-in function
data = input.split(' ')
```


```python
# Print the splitted string
print(data)
```

    ['New', 'Delhi', 'is', 'the', 'capital', 'of', 'India']
