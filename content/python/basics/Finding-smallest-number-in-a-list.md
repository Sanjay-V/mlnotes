```python
# Creating a list 
lst = [53, 26, 14, 91] 
```


```python
# Calling the min() built-in function
data = min(lst)
```


```python
# Printing the minimum element 
print("The Smallest element in a list is:", data)
```

    The Smallest element in a list is: 14
